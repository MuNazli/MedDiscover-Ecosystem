# MedDiscover-Ecosystem

MedDiscover sağlık turizmi platformunun tüm bileşenlerini tek noktada yöneten ana depodur. Veri kaynağından API katmanına ve web deneyimlerine kadar tüm ekipler bu yapı üzerinden koordine olabilir.

## Klasörler
- `meddiscover-data`: JSON veri setleri ile bu verileri tanımlayan şemaların tek doğruluk noktası.
- `MedDiscover-API`: MedDiscover verilerini dış dünyaya açan Node.js/Express tabanlı servis.
- `meddiscover-web`: Hastalara yönelik web arayüzü; şimdilik statik dosyalarla servis ediliyor.
- `meddiscover-admin`: Klinik ve iç ekiplerin kullanacağı yönetim paneli iskeleti.
- `docs`: Mimari, API ve veri modeli dokümantasyonları.
- `scripts`: Yedekleme ve bakım gibi yardımcı betiklerin toplanacağı alan.
