# Sistem Mimarisi

- `meddiscover-data`: JSON veri kümeleri ve şemalar için değişmeyen kaynak.
- `MedDiscover-API`: Node.js + Express kullanarak veriyi REST API üzerinden sunar.
- `meddiscover-web`: Hastaların API'den aldığı bilgilere göre klinik aradığı yüzey.
- `meddiscover-admin`: Klinikler ve iç ekipler için yönetim paneli, operasyonel süreçleri kolaylaştırır.
