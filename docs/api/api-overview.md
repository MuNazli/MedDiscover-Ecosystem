# API Genel Bakış

Şu an yayımlanan temel uç noktalar:
- `GET /api/clinics`: Aktif kliniklerin listesini döner.
- `GET /api/doctors`: Doktor kayıtlarını listeler.
- `GET /api/treatments`: Desteklenen tedavi türlerini sunar.

Filtreleme, ID bazlı detaylar ve diğer gelişmiş uç noktalar ilerleyen sprintlerde eklenecektir.
